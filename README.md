# Idayari

## Description
Idayari est une application web qui permet de créer des agendas.

## Installation
```bash
npm install && npm start 
```

## Fonctionnalités
- Création de compte
    - À partir d'un nom d'utilisateur et d'un mot de passe
- Authentification
- Création d'agendas
